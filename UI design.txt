# ether Portable Synthesizer - UI/UX Design Document

## Design Philosophy

The ether user interface prioritizes **physical control over touch manipulation**, with the smart knob serving as the primary parameter control method. The touch screen functions as a **selection and navigation interface**, while the smart knob provides **precise, haptic-guided parameter adjustment**.

### Core Principles

- **Smart Knob Primary**: Touch selects, smart knob controls with haptic feedback
- **Grid-Based Screens**: 8-10 functions max per screen, clear selection targets
- **Icon-Driven Interface**: Comprehensive iconography for instant recognition
- **Gesture Minimalism**: Primarily swipe navigation, minimal complex touch interactions
- **Color-Coded Organization**: ROYGBIV + Grey system provides instant visual recognition
- **Performance Ready**: All critical functions accessible without looking at screen

## Interaction Model

### Primary Control Flow
```
1. TOUCH: Select function on grid
   â†“
2. Function highlights with cyan border
   â†“  
3. SMART KNOB: Adjust parameter with haptic feedback
   â†“
4. Real-time audio response + visual feedback
   â†“
5. TOUCH: Select next function OR swipe to new screen
```

### Smart Knob Behavior
**Context-Sensitive Haptics**:
```
CONTINUOUS: Filter cutoff, volume - smooth rotation
DETENTS: Pattern selection, BPM - click stops  
SPRING: Pitch bend - return to center
MAGNETIC: Note selection - snap to semitones
TEXTURE: Distortion amount - rough feel
```

**Value Display**:
- **Large numerical display** appears at bottom during knob adjustment
- **Parameter name** shown clearly
- **Range indicators** (min/max values)
- **Unit labels** (Hz, %, dB, etc.)

### Navigation Gestures
```
SWIPE LEFT/RIGHT: Change screen within mode
SWIPE UP/DOWN: Change mode
TAP FUNCTION: Select for smart knob control
TAP SIDEBAR: Switch instrument/utility
LONG PRESS: Context menu (copy/paste/reset)
```

## Animation System

### Minimal Animation Strategy
```
FUNCTION SELECTION: Instant cyan border (no fade)
VALUE CHANGES: Number counter animation (100ms)
SCREEN TRANSITIONS: Slide (200ms) for mode, instant for functions
ICON FEEDBACK: Scale pulse on parameter change (150ms)
LED FEEDBACK: Color transitions (300ms ease)
PLAYBACK: Step indicator moves smoothly (60fps)
```

### Visual Feedback Priority
```
1. SMART KNOB VALUE: Large, clear numerical display
2. SELECTED FUNCTION: Cyan border, no ambiguity
3. INSTRUMENT COLOR: Persistent visual identity
4. PLAYBACK STATE: Clear playhead indicators
5. PARAMETER CHANGES: Icon updates, value animations
```

## Performance Optimization

### Rendering Strategy
```
STATIC ELEMENTS: Icons and grid layout cached
DYNAMIC ELEMENTS: Only values and selection states update
FRAME BUDGET: 16ms target, 60fps for smooth knob response
TOUCH RESPONSE: <50ms from touch to visual feedback
SMART KNOB: <10ms from rotation to value change
```

### Icon Optimization
```
VECTOR ICONS: SVG format for crisp scaling
ICON ATLAS: Single texture for performance
SIZE VARIANTS: 16px, 24px, 32px versions
HIGH CONTRAST: Visible in all lighting conditions
MONOCHROME: Work with instrument color overlays
```

## Accessibility

### Visual Accessibility
```
ICON + TEXT: All functions have both icon and label
HIGH CONTRAST: 4.5:1 minimum contrast ratio
LARGE TARGETS: 168Ã—128px minimum touch areas
CLEAR HIERARCHY: Selected state always obvious
COLORBLIND SAFE: Icons work without color
```

### Physical Accessibility
```

BUTTON FALLBACKS: Key controls for essential functions
MUSCLE MEMORY: Consistent layouts enable blind operation
```
